# 過年祝賀圖片設計專案 (CNY Greeting Image Design Project)

本專案旨在設計一張富含馬年元素與「馬上大賺錢」祝福語的過年祝賀圖片，供使用者於新春期間傳遞祝福。

## 專案內容

- **圖片規格**：130px x 130px
- **設計元素**：
  - 馬年背景圖示
  - 「馬上大賺錢」書法字體
  - 使用者照片合成 (或 AI 生成人物示意)

## 使用方法

1. 下載 `cny_greeting.png`。
2. 若需更換照片，可使用任意修圖軟體將個人照片合成至預留位置。

## 版本資訊

- v1.0: 初始版本，包含 AI 生成之示意圖。
